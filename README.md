## Node.js Password Generator CLI

A simple password generator CLI built with Node.js. This tool allows you to generate secure and random passwords with ease.

### Usage:

1. **Install Dependencies:**
   ```sh
   npm install
   ```

2. **Generate Password:**
   ```sh
   node generate-password.js
   ```
